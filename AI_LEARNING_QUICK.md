---

## 🎯 핵심 개념

**"모든 거래 경험을 학습하여 자동으로 성장합니다"**

```
매수 → 보유 → 매도 → 학습 → 최적화 → 다음 거래 개선 ♻️
```

---

## 🌟 3가지 핵심 기능

### 1️⃣ 📊 경험 축적

모든 거래마다 자동 기록:
- ✅ 진입/청산 가격 & 시간
- ✅ 시장 상황 (RSI, 거래량, 추세 등 14개 지표)
- ✅ 수익/손실 결과

### 2️⃣ 🧠 AI 의사결정

과거 유사 상황 분석:
- 🔍 유사 상황 20건 검색
- 📊 성공률 계산 (예: 75%)
- 🎯 신뢰도 기반 결정

**결과:**
- **STRONG_BUY**: 성공률 ≥ 65%, 강력 매수
- **AVOID**: 성공률 ≤ 35%, 매수 회피

### 3️⃣ ⚙️ 자동 최적화

20회 거래 후 자동 실행:
```
기본 설정:
├─ 손절: 2.0%
└─ 익절: 1.5%

   ↓ AI 학습

최적화 설정:
├─ 손절: 0.8%  ✅ 강화
└─ 익절: 2.1%  ✅ 상향
```

## 💡 실전 예시

### AI 매수 판단

```
현재 상황:
├─ RSI: 28 (과매도)
├─ 거래량: 2.3x (급증)
└─ 추세: 하락

AI 분석:
├─ 유사 상황: 18건
├─ 성공률: 72% (13/18)
└─ 평균 수익: +1.65%

결론: ✅ STRONG_BUY (신뢰도 72%)
```

### AI 매도 판단

```
현재 포지션:
├─ 진입: 50,000,000원
├─ 현재: 50,700,000원 (+1.4%)
└─ 보유: 3분

AI 분석:
├─ 추세 반전 감지 (하락 시작)
├─ 과거 유사 상황 평균: +0.9%
└─ 현재 > 과거 평균

결론: ✅ 즉시 청산 권고
```

### 학습 파라미터 조정

**파일: `src/ai/learning_engine.py`**

```python
self.max_experiences = 10000  # 최대 경험 수
self.min_experiences_for_learning = 20  # 학습 시작 최소값
self.learning_rate = 0.1  # 파라미터 조정 강도
```

### AI 신뢰도 조정

```python
# 매수 판단 (기본: 65%)
if success_rate >= 0.65:
    return 'STRONG_BUY'

# 매도 판단 (기본: 75%)
if confidence >= 0.75:
    return True  # 청산 권고
```

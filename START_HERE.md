# ⚡ 빠른 시작 가이드 (1분 완독)

## 🎯 지금 바로 실행하기!

### 1️⃣ 처음 실행 (초기 설정)
```
setup.bat을 더블클릭
```
→ Python 설치 확인 + 패키지 자동 설치

### 2️⃣ 백테스트로 테스트
```
run_backtest.bat을 더블클릭
```
→ 과거 데이터로 안전하게 전략 테스트

### 3️⃣ 모의투자로 연습
```
run_paper.bat을 더블클릭
```
→ 실시간 데이터 + 가상 거래 (완전 안전)

### 4️⃣ 실거래 (충분한 테스트 후)
```
1. .env 파일 열기 (메모장)
2. UPBIT_ACCESS_KEY, UPBIT_SECRET_KEY 입력
3. TRADING_MODE=live로 변경
4. run_live.bat 더블클릭
```
→ 실제 거래 시작 ⚠️

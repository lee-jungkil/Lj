# ⚡ 초단타 시스템 빠른 참고 (Ultra Scalping Quick Reference)

## 🎯 한눈에 보는 초단타

```
⚡ 30초마다 급등/급락 감지
💰 익절 1% / 손절 0.5%
⏱️ 최대 5분 보유
🎯 최대 2개 포지션
💵 투자: 잔고의 15%, 최대 10만원
```

---

## 🚀 실행

```bash
./run.sh
선택: 3 (live)

# 확인 메시지
🔥 초단타 모드 안내:
   - 5분: 전체 코인 스캔
   - 1분: 일반 포지션 체크
   - 30초: 급등/급락 감지 + 초단타 진입 (최대 2개)
   - 익절: 1% / 손절: 0.5% / 최대 보유: 5분
```

---

```

   가격변동: +3.2%, 거래량: 4.5x

🔥 초단타 매수: KRW-DOGE 
   (급등, 50,000원, 가격변동: +3.2%)
```

```
  ⚡ KRW-DOGE: +0.8% (보유: 30초)
  ⚡ KRW-DOGE: +1.2% (보유: 60초)
```

```

   - 수익률: +1.05%
   - 보유 시간: 65초
```

## ⚙️ 설정 변경

### 더 공격적으로

```python
# src/config.py
'ultra_scalping': {
    'take_profit': 0.015,  # 1.5% 익절 (기본 1%)
    'stop_loss': 0.007,    # 0.7% 손절 (기본 0.5%)
    'max_hold_time': 600,  # 10분 보유 (기본 5분)
}

ULTRA_SCALPING_CONFIG = {
    'max_positions': 3,    # 3개 포지션 (기본 2개)
}
```

### 더 보수적으로

```python
'ultra_scalping': {
    'take_profit': 0.008,  # 0.8% 익절
    'stop_loss': 0.003,    # 0.3% 손절
    'min_price_surge': 0.03, # 3% 이상 급등/급락
    'volume_spike': 4.0,   # 거래량 4배 이상
}

ULTRA_SCALPING_CONFIG = {
    'max_investment_amount': 50000, # 최대 5만원
}
```

## 📈 예상 성과

### 보수적 (50만원 자본)

```

일 거래: 10회
승률: 60%

일일: +2,000원 (+0.4%)
월간: +44,000원 (+8.8%)
```

### 공격적 (100만원 자본)

```
투자: 10만원 × 2개 = 20만원
일 거래: 20회
승률: 65%

일일: +6,650원 (+0.665%)
월간: +146,300원 (+14.6%)
```

## 💡 핵심 팁

### 1. 거래량이 핵심!

```
3배 이상: 신뢰도 높음 ✅
2~3배: 주의 필요 ⚠️
2배 미만: 진입 금지 ❌
```

### 2. 손절 철저히!

```
-0.5% 도달 시 즉시 손절
감정적 복수 매매 금지
5분 쿨다운 후 재진입
```

### 3. 시간대 활용

```
추천:
- 09:00 ~ 11:00 (아침)
- 14:00 ~ 16:00 (오후)
- 21:00 ~ 23:00 (밤)

비추천:
- 03:00 ~ 08:00 (거래량 감소)
```

### 4. 연패 시 대응

```
3연패: 1시간 휴식
5연패: 당일 초단타 중단
전략 재검토
```

## 🔧 문제 해결

### 진입이 없어요

```python
# 감지 조건 완화
'min_price_surge': 0.015,  # 2% → 1.5%
'volume_spike': 2.5,       # 3배 → 2.5배
```

### 손절이 너무 빨라요

```python
'stop_loss': 0.007,  # 0.5% → 0.7%
```

### 익절이 너무 빨라요

```python
'take_profit': 0.015,  # 1% → 1.5%
```
